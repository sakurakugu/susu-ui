# 快速开始

Susu UI 当前优先支持 Vue 3，组件样式依赖 `@susu-ui/theme/style.css` 提供的 CSS 变量。

## 安装

项目还在开发早期，monorepo 内部通过 workspace 引用：

```json
{
  "dependencies": {
    "@susu-ui/theme": "workspace:*",
    "@susu-ui/vue": "workspace:*"
  }
}
```

后续发布到 npm 后，再改为对应版本号。

## 全局注册

在入口文件中引入主题样式，并安装 Vue 插件：

```ts
import { createApp } from 'vue'
import { SusuUI } from '@susu-ui/vue'
import '@susu-ui/theme/style.css'
import App from './App.vue'

createApp(App).use(SusuUI).mount('#app')
```

之后可以在模板中直接使用组件：

```vue
<template>
  <SuButton>默认按钮</SuButton>
  <SuButton type="primary">主要按钮</SuButton>
</template>
```

## 按需导入

也可以只导入需要的组件：

```vue
<script setup lang="ts">
import { SuButton } from '@susu-ui/vue'
</script>

<template>
  <SuButton type="primary">保存</SuButton>
</template>
```

按需导入时也需要在应用入口引入主题样式：

```ts
import '@susu-ui/theme/style.css'
```

## 本地调试

启动 playground：

```bash
pnpm dev
```

修改组件、主题或示例页面后，开发服务会热更新。
