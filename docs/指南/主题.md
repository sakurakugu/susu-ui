# 主题

Susu UI 的 Web 主题分为两条轴：

- 明暗模式：控制背景、文字、边框、阴影等中性色。
- 主题色：控制主色、悬停色、激活色等强调色。

主题能力由 `@susu-ui/theme` 提供，使用前需要引入主题样式：

```ts
import '@susu-ui/theme/style.css'
```

## 切换明暗模式

```ts
import { setThemeMode } from '@susu-ui/theme'

setThemeMode('light')
setThemeMode('dark')
setThemeMode('system')
```

`system` 会跟随系统深色偏好。

如果希望点击按钮时从点击位置扩散切换主题，可以使用 `withThemeTransition`：

```ts
import { setThemeMode, withThemeTransition } from '@susu-ui/theme'

button.addEventListener('click', async (event) => {
  await withThemeTransition(() => setThemeMode('dark'), { event })
})
```

也可以包住多项主题更新，让明暗模式和主题色一起过渡：

```ts
import { setThemeMode, setThemePreset, withThemeTransition } from '@susu-ui/theme'

await withThemeTransition(
  () => {
    setThemeMode('dark')
    setThemePreset('purple')
  },
  {
    point: {
      x: window.innerWidth / 2,
      y: window.innerHeight / 2,
    },
  },
)
```

不支持 View Transition API 的浏览器会直接执行主题更新。

## 切换预设主题色

推荐业务侧优先使用预设主题色：

```ts
import { setThemePreset } from '@susu-ui/theme'

setThemePreset('blue')
setThemePreset('pink')
setThemePreset('green')
setThemePreset('purple')
setThemePreset('orange')
```

可用预设也可以从 `themePresets` 读取：

```ts
import { themePresets } from '@susu-ui/theme'
```

## 自定义主题色

高级场景可以直接设置色相值：

```ts
import { setThemeHue } from '@susu-ui/theme'

setThemeHue(215)
setThemeHue(150)
```

色相值范围是 `0` 到 `359`。传入超出范围的数字会被规范化。

## 初始化主题

可以在应用启动时统一初始化：

```ts
import { initTheme } from '@susu-ui/theme'

initTheme({
  mode: 'light',
  hue: 215,
})
```

## CSS 变量

组件消费 `--su-*` CSS 变量，不直接写死颜色：

```css
.custom-action {
  color: var(--su-color-primary-text);
  background: var(--su-color-primary);
  border-radius: var(--su-radius-md);
}
```

常用变量：

| 变量                        | 说明         |
| --------------------------- | ------------ |
| `--su-color-primary`        | 主色         |
| `--su-color-primary-hover`  | 主色悬停态   |
| `--su-color-primary-active` | 主色激活态   |
| `--su-color-bg`             | 页面背景     |
| `--su-color-bg-elevated`    | 抬升容器背景 |
| `--su-color-text`           | 主文本       |
| `--su-color-text-secondary` | 次级文本     |
| `--su-color-text-muted`     | 次要文本     |
| `--su-color-border`         | 边框         |
| `--su-color-border-strong`  | 强边框       |
| `--su-color-success`        | 成功色       |
| `--su-color-warning`        | 警告色       |
| `--su-color-error`          | 错误色       |
| `--su-color-info`           | 信息色       |
| `--su-radius-xs`            | 极小圆角     |
| `--su-radius-md`            | 默认圆角     |
| `--su-font-size-xs`         | 极小字号     |
| `--su-font-size-xl`         | 超大字号     |
| `--su-shadow-sm`            | 小阴影       |
| `--su-shadow-lg`            | 大阴影       |
